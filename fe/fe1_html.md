# HTML标签

### 列表
- 有序列表
```.html
<ol>
    <li></li>
    <li></li>
</ol>
```
- 无序列表
```.html
<ul>
    <li></li>
    <li></li>
</ul>
```
- 自定义列表
```.html
<bd>
    <dt></dt>
    <dd></dd>
    <dd></dd>
</bd>
```

### 表单控件

- input标签
```.html
<input />
<input type="file" />
```

- textarea标签（文本域）
```.html
<textarea>请输入留言</textarea>
```

- select（下拉框）
```.html
<select>
    <option></option>
    <option></option>
    <option selected='selected'>selected</option>
</select>
```

- form 表单域
```.html
<form action="demo.php" method="post/get">
</form>
```

- html5新标签
```.html
<form action="">
    用户名: <input type='text'/> <br />
    密&nbsp;&nbsp;&nbsp;码: <input type='password'/> <br />
    邮&nbsp;&nbsp;&nbsp;箱: <input type='email' /> <br />
    <input type='submit' />
</form>
```
